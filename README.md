# SARA — Smart Adaptive Response Assistant

SARA is a personalized AI assistant that adapts to user behavior, preferences, and daily routines using contextual memory and feedback-driven recommendations.

## Features
- Personalized AI conversations
- Behavioral memory system
- Daily planner generation
- Context-aware recommendations
- Weekly user insights
- Feedback-driven learning

## Tech Stack
- Python
- Streamlit
- OpenAI API
- JSON Memory Storage

## Vision
To build an intelligent assistant that evolves with the user and becomes a reliable part of everyday life.
